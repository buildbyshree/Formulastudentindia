# Electrical System Overview

This section provides an overview of the electrical and electronic systems used in a Formula Student vehicle. It is intended to help teams understand the overall system architecture and how different electrical subsystems interact with each other.

## Electrical System Categories

The vehicle electrical system can broadly be divided into:

- Low Voltage (LV) System
- Tractive System (TS) — applicable to Electric Vehicles
- Safety and Shutdown Systems
- Communication and Data Acquisition Systems
